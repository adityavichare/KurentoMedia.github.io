FAQ

If you have other than following queries then simply email us.

What are the system requirements?

The Windows Distribution of Kurento Media Server is compatible with all 64-bit variants of Microsoft Windows OS.

I was under the assumption that Kurento Media Server is Open Source - is it not?

While Kurento Media Server is an Open Source project focusing on the Linux Platform, we put a lot of time and effort in curating the project and software dependencies to release stable and tested versions build for the Microsoft Windows operating system. We do offer a package that includes the source code and build system, but unlike the original project – re-distribution is not allowed. If you prefer to work with the original Open Source files - they are available here: https://github.com/Kurento/kurento-media-server

Does the license include commercial support for Kurento Media Server?

We provide a stable binary distribution and build environment of the Kurento Media Server for the Windows Operating system. If you encounter problems that are related to this Windows port please let us know. If you need technical support for the Kurento Media Server itself, there are several third party companies that might be able to provide it incl. Kurento Technologies (see http://www.kurento.org/contact).

Does the license incl. updated versions?

Yes. All future updates are free. We try to closely release new versions after Linux releases are published.
